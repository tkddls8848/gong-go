# 나라장터 통합조회

나라장터의 사전공고와 입찰공고를 조회하고, 기관별 공고 정보를 한곳에서 확인할 수 있도록 돕는 웹 애플리케이션입니다.

## 서비스

이 앱은 [Cloudflare Pages](https://pages.cloudflare.com/)를 통해 호스팅되며, 서비스 주소는 [https://gong-go.pages.dev/](https://gong-go.pages.dev/)입니다.

## 참고 API 문서

- [나라장터 사전규격 공개 API](https://www.data.go.kr/data/15129394/openapi.do)
- [나라장터 입찰공고 API](https://www.data.go.kr/data/15129437/openapi.do)
